# specify
我想实现的三个核心功能是
1. 完整解析pptx和ppt文件的所有内容，除去宏定义以外，其他内容都要从pptx或ppt文件中提取为xml.json中。
解析过程要求按照ECMA-376 office open XML标准里PresentationML部分的内容进行完整映射，将xml文件里定义的每一个属性都转为json属性。在转换过程中，要在代码里和Conversion.md里写清楚每一个被转换的属性是什么意思。
2. 将第一步生成的xml.json文件转为PPTist这个功能可以使用的pptist-Conversion.json文件，要求按照PPTist里slides.ts的完整定义进行转换。导入到PPTist的json文件格式应该参考pptist.json的格式，有theme对象和slides数组，theme对象的属性参照slides.ts里SlideTheme的定义，slides数组里每个元素都是一个slide对象，每个slide对象的具体属性参照slides.ts里关于Slide的定义。且各种类型都要对应到位。
3. 将pptist-Conversion.json文件使用和PPTist里一样的encrypt函数进行加密，加密后的文件名为pptist-Conversion.pptist 可以直接导入到PPTist中使用。
4. 实现一个简单的web服务，我上传pptx文件，服务会返回pptist-Conversion.json 和 pptist-Conversion.pptist 两个文件。
